# WiDS 2025 Midterm Submission

This repository contains my midterm submission for the WiDS 2025 project on predicting stock volatility using financial news data.

## Work Completed Till Midterm

- Collected real-world financial news using RSS/XML feeds
- Parsed and structured news data into a tabular format
- Cleaned and standardized publication timestamps
- Collected historical stock price data using the yfinance API
- Identified trading and non-trading days
- Merged news data with stock price data
- Performed basic exploratory data analysis with simple visualizations

## Repository Contents

- `midterm_submission.ipynb` – Jupyter Notebook containing all code and analysis
- `news_raw.csv` – Raw news data collected from RSS feeds
- `news_cleaned.csv` – Cleaned and standardized news data
- `merged_midterm_data.csv` – Final merged dataset with trading day indicators
- `stock_data.csv` – Stock price data used for merging
